클래스란 ? class를 사용하면 객체를 생성하기 위한 템플릿을 정의 할 수 있다. 코드의 재사용성, 유지보수성을 높일 수 있다. class를 사용하면 객체 지향 프로그래밍의 주요 개념인, 캡슐화, 상속, 다향성 등을 활용 할 수 있다. 기본 문법 class className { 생성자 메서드 constructor(prarametres){ 초기화 코드 } 메서드 정의 mathodName(){ 메서드 로직 } } constructor : 클래스의 인스턴스를 생성 할 때 호출 되는 특별한 메서드이다. 주로 객체의 초기 상태를 설정 하는데 사용 메서드 : 클래스 내부의 정의 된 함수, 해당 클래스의 인스턴스가 수핼 할 수 있는 동작이 정의 됨
