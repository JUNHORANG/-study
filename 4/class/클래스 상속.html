상속 클래스는 다른 클래스를 상속 받는다. 상속을 통해 기존 클래스의 기능을 확장 하거나 재사용할 수 있다. 기본 구조 class childClass extands parentClass{ constructor(...args){ super(...args)// 부모 클래스의 생성자 호출 자식 클래스의 초기화 코드 } 자식 클래스의 메서드 정의 } extends : 클래스 상속을 구현하기 위해 사용하는 키워드, 상위 클래스의 속성과 메서드를 하위 클래스가 상속 받아 재 사용하거나 확장 할 수 있게 한다. super : 부모 클래스의 생성자를 호출, 자식 클래스의 생성자 반드시 호출
